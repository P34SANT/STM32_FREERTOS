# FreeRTOS on STM32

A collection of small FreeRTOS practice projects and experiments on STM32.

Each folder focuses on a specific RTOS feature such as:
- Task management
- Priorities and delays
- Queues and queue sets
- Semaphores and mutexes
- Event groups
- Software timers
- Task notifications
- ISR communication

The goal of this repository is to build hands-on experience with FreeRTOS concepts on embedded systems.

---
Board      : STM32F407VET6 Dev Board  
IDE        : STM32CubeIDE  
Framework  : STM32CubeMX + FreeRTOS
